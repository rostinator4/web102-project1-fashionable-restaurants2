# Web Development Project 1 - Fashionable Restaurants

Submitted by: Rostislav Dadazhanov

This web app: using css grid instead of table (it caused cards to be of different size), I implemented a simple website that lists
10 fashionable restaurants in Chicago. Additionally, I provided buttons on the bottom of a card to link each website. 

Time spent: 4 hours spent in total
